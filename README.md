# MoneyPath - お金の道筋を可視化するWebアプリ

## コンセプト
明るい将来が見えるお金管理アプリ。借金返済や目標貯金額達成までのロードマップを視覚的に表示。

## セットアップ

詳細なセットアップ手順は [SETUP.md](./SETUP.md) を参照してください。

クイックスタート：

```bash
# Docker Desktopを起動後
npm run docker:dev

# ブラウザで http://localhost:3000 にアクセス
```

## 主要機能
- 現状の収支・負債の入力
- 目標設定（借金完済日、貯金目標額など）
- 達成までのロードマップ自動生成
- 月々の必要ペース計算
- What-ifシミュレーション（節約したら？収入増やしたら？）
- 進捗の視覚化（グラフ、プログレスバー）

## 技術スタック
- **Framework**: Next.js (App Router) + TypeScript
- **Styling**: Tailwind CSS
- **Database**: MySQL
- **ORM**: Prisma
- **グラフ**: Recharts
- **環境**: Docker Compose

## ターゲットユーザー
- 借金返済中の人
- 目標貯金額がある人
- お金の不安を可視化したい人
- 年齢・職業不問

## 差別化ポイント
- 単なる家計簿ではなく「ゴールから逆算」する設計
- 過去の記録より「未来への道筋」にフォーカス
- 希望が見える前向きなUI/UX
- シンプルで直感的な操作性

## MVP（最小構成）で実装する機能
1. 収入・支出・負債の入力
2. 目標設定（金額と期限）
3. 達成予定日の計算・表示
4. 簡単なグラフ表示
5. 月々の必要ペース表示
